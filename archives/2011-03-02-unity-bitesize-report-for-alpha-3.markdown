### Other Contributors this Week

-   We have a new contributor this week, Marco Bisaro ﻿has fixed 
    Bug [\#713789](https://bugs.launchpad.net/ubuntu/+source/unity/+bug/713789)
    – Upper left Ubuntu logo has different background color than panel.

-   Olivier Sauder’s work on keybindings for the launcher has landed

-   Mathew Rasmus has made it so the launcher uses proper getter/setter
    for the Launcher’s member-var\_launcher\_action\_state.

-   This could be you! See below!

### General Unity Fixes and New Stuff

What a crazy week! Double dose of unity (Thursday and Tuesday)! We also
got a new compiz with a full stack rebuild/update. That totalize more
than 23 updates within the week. As you can see, we got a lot of fixes
and enhancements:\
[\[https://launchpad.net/unity/3.0/3.4.6\](https://launchpad.net/unity/3.0/3.4.6)\
](https://launchpad.net/unity/3.0/3.4.6)[\[https://launchpad.net/unity/3.0/3.6\](https://launchpad.net/unity/3.0/3.6)](http://www.omgubuntu.co.uk/2011/03/unity-alpha-3-dash-lands-and-bitesize-progress-extravaganza/onclick=)

### Other Highlights

-   Most of known common crashers for unity have been tackled

-   Lot of accessility improvments

-   Lot of launchers fix and most of “false show” positive or “launcher
    being crazy” on intellihide mode fixed

-   Drag and drop from the application and file places to the launcher
    is now functional (but still in beta
    Funny ![:)](http://www.omgubuntu.co.uk/wp-includes/images/smilies/icon_smile.gif) )

-   New shortcuts with the Super key:

    -   tap super open the dash

    -   Keeping super press will get some shortcuts showing: Super + to
        get active window of an application or show the places/ws
        switcher/trash

    -   if you add Shift to the dance, it will open a new instance of an
        application instead of getting an existing one

    -   Key navigation to all elements of the launcher now, and
        quicklist! Alt + F1 showing the launcher when hidden.

-   Rework fade effect animation when hovering the bfb (the ubuntu logo)
    Still some known crasher on the unity window decorator. It’s tracked
    and under debugging right now. Hope we can get a fix after alpha3
    on upgrade.

How to Get Involved
-------------------

1.  Get the Code

Follow the [Step by Step
Instructions](http://unity.ubuntu.com/getinvolved/) and [Wiki
Page](https://wiki.ubuntu.com/Unity/Bitesize). This will get the code
from Launchpad, set up your development environment, and getting you
used to the Launchpad workflow.

### 2. Pick a bug

This week we want to focus on these bugs, pick one:

-   [Bug 718889](https://bugs.launchpad.net/unity/+bug/718889) - 
    Launcher does not hide/unhide on Expo

-   [Bug 660010](https://bugs.launchpad.net/unity/+bug/660010) – No
    ‘safely remove’ option is present in the unity menu when a usb disk
    is inserted

-   [Bug 718885](https://bugs.launchpad.net/unity/+bug/718885) –
    Launcher responds to click off from a quicklist

-   [Bug 721121](https://bugs.launchpad.net/unity/+bug/721121) – Icon in
    Launcher should be home folder icon

-   [Bug 718885](https://bugs.launchpad.net/unity/+bug/718885) – (NEW!)
    Launcher responds to click off from a quicklist

-   [Bug 725529](https://bugs.launchpad.net/unity/+bug/725529) – (NEW!)
    ﻿Double-click on panel to unmaximize only works in right half

-   [Bug 724727](https://bugs.launchpad.net/unity/+bug/724747) – (NEW!)
    Super key shortcut overlay not with umlauts, greek/cyrillic letters

Here’s the [full list](http://goo.gl/tiheb) if you want to find more,
feel free to just grab one, assign it to yourself, mark it in progress
and get started. I’ll be picking a new selection for each blog post each
week, but the list is always changing, so you can always just dive into
the list and snag one.

### 3. Fix your bug and then get your code into Unity

Don’t worry we won’t leave you hanging, you can get a-hold of a Unity
developer through many different ways:

-   ayatana on freenode IRC during European and American workdays. Or you can post to [the mailing list](https://launchpad.net/~ayatana-dev) if you have a question. {#ayatana-on-freenode-irc-during-european-and-american-workdays-or-you-can-post-tothe-mailing-listhttpslaunchpadnetayatana-devif-you-have-a-question}
    ================================================================================================================================================================

-   We also have weekly IRC Q+A for any developer who wants to dive in
    and ask a Unity developer. 7pm-8pm UTC (That’s 2pm EST) every
    Friday!

-   Mention session in developer week.
