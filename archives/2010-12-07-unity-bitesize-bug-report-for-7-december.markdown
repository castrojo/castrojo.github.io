<div class="entry-content" itemprop="articleBody">

As I mentioned [last
week](http://castrojo.tumblr.com/post/2084859426/getting-started-working-on-unity)
we’ve started a campaign for helping people get started on fixing
bitesize bugs in Unity. Jono
[blogged](http://www.jonobacon.org/2010/12/06/get-started-hacking-on-unity-bitesize-bugs/)
about this as well. As it turns up, we’ve got our first bitesize
contributor today!

Jamal Fanaian has fixed [Bug \#683241: Recycle bin icon is empty when
there are items in the
bin](https://bugs.launchpad.net/unity/+bug/683241). Jamal is a developer
working on Flickr. He passes along “I love Ubuntu and open source
software, so my only real hobby has become working on open soruce. Other
than that, I like playing with hardware, although recently I have found
less time to do so.”

Basically the trash icon showed up empty even if you had stuff in it.
Jamal first grabbed the code, then fixed the bug, and [then pushed it to
launchpad](https://code.launchpad.net/~jamalta/unity/683241-recyclebin-icon).

![](http://farm5.static.flickr.com/4009/4651051589_073cdd654a_m.jpg)

Photo by [brittav](http://www.flickr.com/photos/sanguinetangox/).

From there he [submitted it for
review](https://code.launchpad.net/~jamalta/unity/683241-recyclebin-icon/+merge/42917)
in Launchpad. In this specific example Alex Launi has started the review
process. Now we’re cooking. So, how can you get started? First let’s
have a look at [the
list](https://bugs.launchpad.net/ubuntu/+source/unity/?field.omit_dupes=on&field.tag=bitesize)
and look at the progress people have made this week.

-   ~~Bug 677577 - \[Clicking on a launcher icon does not raise most
    recent window\](https://bugs.launchpad.net/unity/+bug/677577)~~
-   Fixed!

    -   Bug 646740 - [indicators are
        mis-aligned](https://bugs.launchpad.net/unity/+bug/646740)

    -   ~~Bug 683241 - \[Recycle bin icon is empty when there are items
        in the bin\](https://bugs.launchpad.net/unity/+bug/683241)~~

-   Fixed!

    -   Bug 683466 - [There are 2 Quit menu options in
        Quicklist](https://bugs.launchpad.net/unity/+bug/683466)

    -   Bug 599716 - [having “unity –replace” would be
        nice](https://bugs.launchpad.net/unity/+bug/599716)

    -   Bug 681428 - [scrolling does not work on the sound
        menu](https://bugs.launchpad.net/unity/+bug/681428) - Music
        lovers everywhere would appreciate a fix here.

New bitesize bugs this week (they might be older bugs but new to
bitesize):

-   [Bug
    686591](https://bugs.launchpad.net/ubuntu/+source/unity/+bug/686591) -
    unity-panel-service should be autorestarted by unity when crashing -
    A fix for this one would be most welcome, as it’s annoying!

-   [Bug
    677594](https://bugs.launchpad.net/ubuntu/+source/unity/+bug/677594) -
    Workspace switcher useless with one workspace

-   Armor Nick seems to be doing the initial investigation here.

    -   [Bug
        686182](https://bugs.launchpad.net/ubuntu/+source/unity/+bug/686182) -
        Unity launchers run multiple copies of program if clicked
        multiple times before the program loads

    -   [Bug
        599716](https://bugs.launchpad.net/ubuntu/+source/unity/+bug/599716) -
        having an unity binary would be nice

Getting Involved: Instructions for getting started are [available
here](http://unity.ubuntu.com/getinvolved/%20) 

For more information check out the wiki page:
[\[https://wiki.ubuntu.com/Unity/Bitesize\](https://wiki.ubuntu.com/Unity/Bitesize)](https://wiki.ubuntu.com/Unity/Bitesize)
