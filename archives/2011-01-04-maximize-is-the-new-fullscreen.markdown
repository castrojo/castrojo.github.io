Here’s something I discovered yesterday as I was tooling around on my
home server while using Unity. This is what happens when you maximize a
terminal in Unity Alpha in Natty (Assuming you have autohide on, it’s
not turned on by default yet but it will be, [here’s
how](http://askubuntu.com/questions/9865/will-the-unity-launcher-auto-hide/9881#9881)):

![](http://media.tumblr.com/tumblr_leia81DJco1qb5bmy.png)

([Bigger pic](https://dl.dropbox.com/u/5720/unity-awesome-for-cli.png))

If this was normal Ubuntu you’d have another panel AND the titlebar.
This is all now fused into one out of the way thing, saving pixels and
getting out of my way. 

I know what you’re thinking. Endless combinations of terminal tabs
and/or Terminator splits and byobu. I might almost want to be a
sysadmin. Here’s gedit:

![](http://media.tumblr.com/tumblr_leiaobKZEL1qb5bmy.png)

([Bigger pic](https://dl.dropbox.com/u/5720/Workspace%201_002.png))

This is real useful when I’m working on something, I need my desktop to
be out of the way. Obviously there’s a lot of work to be done in this
area, but it’s coming together, stay tuned for a Unity progress report
later today.
