<div class="entry-content" itemprop="articleBody">

The dynamic duo of Calvin Gaisford and Boyd Timothy have unleashed
[Tasky](http://code.google.com/p/tasky/wiki/ScreenShots) onto the world,
which they worked on during Novell’s Hack Week.

Tasky is a simple TODO app for the Linux desktop, though I think the
real awesome bit is the integration with [Remember The
Milk](http://www.rememberthemilk.com), an online task list. I really
like RTM’s features, especially the Firefox plugin for Gmail, which is
really fantastic. The reason I like tasky is that I can have a desktop
application that talks to RTM which I can “Always on Top” on my second
monitor and have it in my face all the time.

[![](http://lh3.google.com/jorge.castro/R8dxmWca1TI/AAAAAAAAEmk/IxwF5K7JQdk/s288/Screenshot.png)](http://picasaweb.google.com/jorge.castro/RandomCrap/photo#5172227600732050738)

Like any new project, it is very young and missing features, but if you
use RTM and check out Tasky, you will quickly realize the potential here
for a great tool. For example, someone has already written a backend for
Evolution tasks. I threw up some ideas for future work
[here](http://groups.google.com/group/tasky/browse_thread/thread/a5fd28d0a1488e25).

Join the [tasky google group](http://groups.google.com/group/tasky) for
discussion; get the code
[here](http://code.google.com/p/tasky/source/checkout), and feel free to
jump in.

Edit: Misspelled Calvin’s name, fixed. I guess I owe him a beer for that
one.
