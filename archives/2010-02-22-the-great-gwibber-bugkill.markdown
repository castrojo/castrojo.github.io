For the past month or so Ryan and Ken have been basically rewriting
chunks of Gwibber, the social networking client that we’re including in
Lucid. As we’re passing Feature Freeze and getting into Alpha3 and soon
Beta it’s time to start ratcheting up our bug fixing efforts.
Unfortunately we have tons of bug reports from older versions of
Gwibber. Check out [the list
here](https://bugs.edge.launchpad.net/ubuntu/+source/gwibber).

We need volunteers to go through these bugs and see if they’re still a
problem. And if not, resolve them.

If you want to [Adopt
Gwibber](https://wiki.ubuntu.com/BugSquad/AdoptPackage) and help clean
house and tidy things up it would really help Ken and Ryan out so they
can concentrate on fixing the bugs that are affecting current users and
not weeding out old crashers that probably don’t apply anymore. If you
find a bug and want to actually fix it, then please fix it and propose a
merge, the team is getting much better at doing reviews.

My current gwibber instance has been running for about 2 weeks and it
hasn’t crashed, so we’re definitely improving when it comes to
reliability. Now it’s up to us to collectively filter the wheat from the
chaff to bring the level of polish people will expect for Lucid.
