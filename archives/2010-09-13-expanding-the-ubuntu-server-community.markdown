<div class="entry-content" itemprop="articleBody">

I’ve put out a [call for
help](https://lists.ubuntu.com/archives/ubuntu-server/2010-August/004577.html) for
server interested folks to start working with upstreams. On top of that
we’ve got a new contributor on Community Team at Canonical, [Ahmed
Kamal](http://www.jonobacon.org/2010/07/15/ahmed-kamal-joins-the-horsemen/),
who’s grabbed the bull by the horns and helped step up to grow the
server community (cloud and otherwise), along with the always amazing
[Dave Walker](http://blog.daviey.com/). 

Now that 10.04.1 is out the door here are some of the reasons I think
Ubuntu Server rocks, add your own!

-   Cloud, [out of the box](http://www.ubuntu.com/cloud). Both private
    and public. Wether you dig the EC2 work spearheaded by Eric Hammond
    (with thanks to Scott Moser for continuing the work), or you want to
    run your own via [Eucalyptus](http://www.eucalyptus.com/), you can
    do that. Did you know you can customize your -server images [on
    boot](http://ubuntu-smoser.blogspot.com/2010/03/introducing-cloud-inits-cloud-config.html)?

-   Ubuntu Enterprise Cloud has been focused on getting you from ISO to
    cloud in about 5 questions and 20 minutes.

-   Did I mention we give you Puppet out of the box? And we don’t just
    ship it, we participate with the folks at Puppet Labs to deliver you
    the best Puppet experience … and we take that one step further when
    it comes to /etc, by [integrating it
    all](https://wiki.ubuntu.com/PuppetEtcKeeperVCSIntegration) with
    etckeeper and bzr.

-   Check out tomcat6-instance-create, easily allows one to setup
    multiple separate tomcat instances for any user, while still
    benefiting from distro upgrades.

-   Little nice things, like a full blown window manager for the CLI in
    [Byobu](http://launchpad.net/byobu), command-not-found, an entire
    easy to use LAMP stack in [one
    command](http://ubuntu.stackexchange.com/questions/34/whats-the-easiest-way-to-set-up-a-lamp-stack-on-ubuntu).

-   Amazing [ClamAV
    support](http://castrojo.wordpress.com/2010/04/16/clamav/).

-   An [ever growing
    list](https://edge.launchpad.net/~ubuntu-server-edgers) of daily
    builds of server upstreams to help you test those dev boxes,
    maintained by Chuck Short.

-   All wrapped up with [great security
    support](https://wiki.ubuntu.com/Security/Features), with apparmor
    profiles for cupds, openldap, mysql, bind, clamav, tcpdump, dhcp
    (client and server), ntpd, libvirtd (and guests), and apache2.

You’ll also find Ubuntu Server folks collaborating with Debian on
tomcat6, puppet, postfix, bind9, apache2, and php5, to mention a few.

Whew, what a list! What’s your favorite server feature? Like server
stuff? Check out the [Official Ubuntu Server Book, Second
Edition](http://www.amazon.com/Official-Ubuntu-Server-Book-2nd/dp/0137081332/ref=sr_1_1?ie=UTF8&s=books&qid=1282449673&sr=8-1).
If you are not familiar with the first edition, the book is authored
by [Kyle Rankin](http://www.linuxjournal.com/users/kyle-rankin), one of
the columnists for [Linux Journal](http://www.linuxjournal.com/).
