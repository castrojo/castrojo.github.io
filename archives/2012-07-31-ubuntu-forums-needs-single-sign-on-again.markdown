<div class="entry-content" itemprop="articleBody">

A while back [I asked
around](http://www.jorgecastro.org/2012/01/20/ubuntu-forums-needs-single-sign-on-looking-for-a-php)
for a PHP developer to help implement single sign on on the Ubuntu
Forums.

It ends up our ISD team had cobbled together something to make it work,
but it wasn’t very maintained and loved, and broke with vBulletin 4.x,
so we’ve been hobbling along on 3.x.xsomething something.

Anyway, the code to make SSO work has been Open Sourced by Canonical,
sorry it took us so long:

-   <https://launchpad.net/vbulletin-openid-integration>
-   <https://launchpad.net/vbulletin-launchpad-integration>

The bummer is we still don’t have it working with vBulletin 4.x. So
we’re looking for someone to step up and see if we can make this work.
Ideally someone who has worked with VB before and has some knowledge of
how SSO works. Canonical can provide access to a license, so if you have
any leads on someone who can do this, or if you can do it, let me know
and we’ll get started!
