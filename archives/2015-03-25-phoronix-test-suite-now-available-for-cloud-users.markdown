<div class="entry-content" itemprop="articleBody">

Phoronix [ran an article
today](http://www.phoronix.com/scan.php?page=news_item&px=Ubuntu-Cloud-JuJu-Benchmark)
highlighting our work on making a [Juju
charm](https://github.com/phoronix-test-suite/phoronix-test-suite/tree/master/deploy/juju)
for the [Phoronix Test Suite](http://www.phoronix-test-suite.com/).

![](http://www.jorgecastro.org/images/pts-logo-60.png)

This enables people to easily deploy pts to any cloud that we support,
so AWS, Azure, Joyent, HP Cloud, GCE, onlinelabs.net, digitalocean, and
of course bare metal and OpenStack. So now you can try any cloud
provider and run your own benchmarks to see which one works best for
you. With Windows, Debian, and CentOS arriving in Juju soon we’ll have
your cross-OS needs covered as well.

Marco Ceppi and Adam Israel will be adding some more fixes and features
over the next few days, you can follow along (and help if you’d like) in
the [Phoronix upstream
git](https://github.com/phoronix-test-suite/phoronix-test-suite/tree/master/deploy/juju).
Many thanks to Michael Larabel for giving us an area in the pts git repo
to do this work!
