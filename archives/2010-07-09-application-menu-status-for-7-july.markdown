Hi everyone,

Another week, another release of the new menu for UNE. This week
indicator-appmenu did not have any releases, as the bugs reported have
been in appmenu-gtk. Cody Russell has released 0.1.1 of appmenu-gtk,
which is in Maverick and currently building for the Lucid PPA. Here are
the highlights:

-   Bullets should now appear as bullets instead of checkmarks
-   Fix errors in GIMP

Full changelogs:
<https://launchpad.net/ubuntu/maverick/+source/appmenu-gtk/0.1.1-0ubuntu1>

Note that if you are using Unity that there is a mutter bug that breaks
submenus that might be confused for an appmenu bug, please see:
<https://bugs.launchpad.net/ubuntu/+source/unity/+bug/592949>

The menu itself is in pretty good shape right now, unless someone has
any objections we’d like to turn the double menus off in the
applications for next week’s release. Special thanks to Hernando Torque
for his detailed bug reporting for the menus!
